Link Appscript

Gunakan custom domain domain.com/namaweb untuk Web App Google Script Anda tanpa pusing memikirkan Iframe!

Wajib Tambahkan Kode Ini di Code.gs

return HtmlService.createHtmlOutputFromFile('Index')
  .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);

Tanpa kode ini, Google akan memblokir web app Anda untuk ditampilkan di dalam Iframe.
